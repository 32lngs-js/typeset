Typography is the interface

Most of a product's voice lives in its type. This page is here so you can scrub it.

This is a sample host page. Open the TypeSet overlay (it loads automatically here), then click any text element to inspect and edit its live typography — size, weight, line height, letter spacing, width, and family.

Try the new bits: shift-click several elements to select them at once, or drag a rectangle across a region to marquee-select. Then scrub a slider and watch every selected element change together. Hit Copy CSS and you get one selector block per element.

What you can change

  • Size, weight, line height, and letter spacing
  • Column width (measure) via the Layout slider
  • Font family, previewed in each face
  • Alignment and X / Y position

Everything you do is a live inline style, so it resets on reload. The Copy button is how you take the values back to your real stylesheet.
